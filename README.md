<h1 align="center">Tokiwa Launcher</h1>

<em><h5 align="center">Minecraft カスタムランチャー by Tokiwa Haruto</h5></em>

<p align="center">Modのインストールも、Javaの設定も、全部おまかせ。ワンクリックでサーバーに接続できます。</p>

## 機能

* 🔒 **Microsoftアカウント認証対応**
  * Microsoft (OAuth 2.0) による安全なログイン
  * 複数アカウントの管理・切り替え
* 📂 **自動Mod管理**
  * Fabric + Sodium / Lithium / Iris Shaders / Fabric API を自動インストール
  * にゃんクラサーバー向け Simple Voice Chat 対応
  * ファイルの整合性を起動前に自動検証
* ☕ **Java自動検出**
  * 互換性のあるJavaが見つからない場合は自動でインストール
* 🖥️ **複数サーバー対応**
  * get0 / meme / private / Haruto / にゃんクラ
  * サーバー切り替えはワンクリック
  * プレイヤー数のリアルタイム表示
* 🌐 **日本語UI**
* 🔄 **自動アップデート**

## インストール

1. 上のリンクからEXEをダウンロード
2. 実行してインストール
3. Microsoftアカウントでログイン
4. サーバーを選んでプレイ！

## コンソール

デバッグコンソールを開くには:

```
Ctrl + Shift + I
```

## 開発

### 必要環境

* [Node.js](https://nodejs.org/) v22

### セットアップ

```console
git clone https://github.com/NvdaAnsel/TokiwaLauncher.git
cd TokiwaLauncher
npm install
```

### 起動

```console
npm start
```

### ビルド

```console
npm run dist:win
```
